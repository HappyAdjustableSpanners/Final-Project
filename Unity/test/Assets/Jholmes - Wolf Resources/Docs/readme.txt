Wolf - Model, Rig and Animation.
----------------------------------

A demo scene can be found under ExampleScene/ExampleScene.unity. This scene showcases the different animations that can be applied to the wolf.
The animation can be changed by using the AnimateWolf.cs script which is attached to each wolf. Simply select the animation from the drop-down
menu on the script.

The wolf prefab can be found under Assets/Prefabs/Wolf.
